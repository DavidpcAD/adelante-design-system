# UX/UI Guidelines

Este documento define las reglas de uso del sistema de diseño.

## Principios

- Consistencia visual en todas las aplicaciones
- Uso obligatorio de componentes del design system
- Accesibilidad básica en formularios, tablas y botones
- Uso correcto de colores, espaciados y tipografía
- Reutilización antes de crear nuevos componentes

## Reglas generales

- No crear botones con estilos diferentes fuera del sistema
- No usar colores fuera de los tokens definidos
- Todos los formularios deben seguir la misma estructura visual
- Las tablas deben mantener encabezados, espaciado y estilos consistentes
- Los íconos deben salir de los assets oficiales

## Referencias

- Figma design system
- Tokens de diseño
- Componentes React
- Assets oficiales
